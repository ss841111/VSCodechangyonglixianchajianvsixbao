## 使用方法

1. **下载插件**：从本仓库中下载所需的vsix文件。
2. **安装插件**：
   - 打开VS Code。
   - 进入扩展管理页面（快捷键：`Ctrl+Shift+X`）。
   - 点击右上角的“...”按钮，选择“从VSIX安装...”。
